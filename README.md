# TODO app API
## Description
TODO app API is one of two parts of the TODO list project. This repository contains API to manage user accounts, their lists and tasks.
## Author
[@Blindeen](https://www.github.com/Blindeen)
## How to run
### Clone
```bash
git clone https://github.com/Blindeen/todo-api.git
```
### Directory
```bash
cd todo-api
```
### Environment
```bash
python -m venv venv
source venv/bin/activate
```
### Packages
```bash
pip install -r requirements.txt
```
### Run server
```bash
python manage.py runserver
```
